# AI Star

現代許多家庭是隔代教養家庭或雙薪家庭
以手機、平板陪伴小孩 有礙於身心發展
本作品針對2-8歲小朋友
藉由誘導孩子說出心中想法
並透過對應的故事抒發心情

---
## tool
* RPi.GPIO
* import speech_recognition
* wave
* jieba
* pyaudio
* sqlite3
* E-hownet api

## hardware
* Raspberry Pi 3 Model B+
* ReSpeaker 2-Mics Pi HAT
